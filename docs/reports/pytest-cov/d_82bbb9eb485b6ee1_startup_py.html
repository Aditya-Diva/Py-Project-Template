28 """Env Loading Class - Meant to act as a single blueprint for loading env

42 # Get the initial env to understand if we should run in debug mode 

43 try: 

44 debug_mode: bool = EnvLoader(debug_mode=os.environ["DEBUG_MODE"]).debug_mode 

45 except ValidationError as valerr: 

46 sys.stderr.write("Env vars values are not valid. Please check Env Variables!")
